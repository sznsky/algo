# algo-study

#### 介绍
算法学习-极客时间算法训练营。极客时间李煜东老师的算法课，本项目记录算法学习的作业。跟着大牛学习算法。
包括课上老师讲的题目，还有安排的作业，自己做的习题。

#### 算法内容
算法：数组，链表，图，栈，动态规划等等
